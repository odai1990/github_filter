# GitHub Filter

## Project Structure

```
src -- >
            Api--> The Api calling for all project
            ----------------------------------------
            Assets-->
                        img --> All images used in project
                        scss --> Basics css for project
            -----------------------------------------

            Components-->Contains the shared components
                            CustomButtons
                            CustomCard
                            CustomNotification
                            CustomSearch

            -----------------------------------------
            Layout--> Where to add header , footers ...etc
                        Header
            -----------------------------------------
            Pages--> Where to add you pages
                    FilterPage
            -----------------------------------------
            App --> The root of Project
```

## Features

1- Hosted in vercel [https://github-filter.vercel.app/]
2- Full responsive
3- Search suggestions while typing
